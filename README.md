Guess The Emoji

An interactive emoji guessing game built with JavaScript where players must identify a randomly selected emoji from a collection of five within two attempts. The game provides feedback after each guess and allows users to replay multiple rounds.

Features

    ->Random emoji selection from a predefined set
    
    ->Two-attempt guessing challenge
    
    ->Dynamic feedback for correct and incorrect guesses
    
    ->Replay / reset functionality
    
    ->Simple and responsive UI

Technologies Used

    ->HTML
    
    ->CSS
    
    ->JavaScript (DOM manipulation & event handling)

How to Play

    ->Observe the emoji collection displayed on the screen.
    
    ->Enter your guess for the hidden emoji.
    
    ->You have two attempts to guess correctly.
    
    ->Receive feedback after each attempt.
    
    ->Use the Again button to start a new round.
